# 秋蒂桌宠说明文档

## 介绍

该桌宠是基于秋蒂虚拟主播人物所开发的。软件下载需要去B站关注秋蒂，找秋蒂获得软件压缩包即可！用户下载安装后就可以选择秋蒂作为自己的桌宠，与她进行聊天和互动，还可以查看日程表了解秋蒂直播的日程。

## 功能

- [x] 支持 Live2d moc/moc3

- [x] 支持 拖拽模型(单屏/双屏)

- [x] 开机自启动

- [x] 语音口型同步

- [x] 日程表

- [x] ChatGPT聊天
