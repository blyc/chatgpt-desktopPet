# vits-desktopPet

基于Electron开发 Live2D模型的桌面宠物

## 开发 (Develop)

- 使用 `npm` 安装：

  ```shell
    npm install
  ```

- 使用 `cnpm` 安装：

  ```shell
    npm install -g cnpm --registry=https://registry.npmmirror.com
    cnpm install
  ```

- 启动方式：

  ```shell
    npm run start
  ```

- 打包方式

  ```shell
    npm run dist-win64
  ```

## 功能 (Function)

- [x] 支持 Live2d moc/moc3
- [x] 支持 拖拽模型
- [x] 切换 live2D模型
- [ ] 日程表
- [ ] ChatGPT聊天
