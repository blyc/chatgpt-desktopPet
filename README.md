# vits-desktopPet

基于Electron开发 Live2D模型的桌面宠物

## 开发 (Develop)

- 使用 `npm` 安装：

  ```shell
    npm install
  ```

- 使用 `cnpm` 安装：

  ```shell
    npm install -g cnpm --registry=https://registry.npmmirror.com
    cnpm install
  ```

- 启动方式：

  ```shell
    npm run start
  ```

- 打包方式

  ```shell
    npm run dist-win64
  ```
